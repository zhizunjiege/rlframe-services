# rlframe-services

## Protos

Use below command to build protos:

For Linux:

```bash
./tools/build-protos.sh
```

For Windows:

```bat
./tools/build-protos.bat
```

## Tests

Use below command to run tests:

For Linux:

```bash
./tools/unittests.sh
```

For Windows:

```bat
./tools/unittests.bat
```
